<h3>DotNet</h3>

**Platform Creation:**

``` DotNetPlatform.New(PlatformConnectorType); ```

**Platform Types:**

- **RPC:** ``` Platform myNewPlatform = DotnetPlatform.New(PlatformConnectorType.RPC); ```
- **Sharpion:** ``` Platform myNewPlatform = DotnetPlatform.New(PlatformConnectorType.Sharpion); ```
